# Posters Zaska y Accion

Biblioteca de referencia para posters Zaska y Accion generados con los prompts de Notion.

Convencion:
- Guardar cada poster como `[slug].png`.
- Usar nombres en minusculas, sin tildes y con guiones.
- Ejemplos: `the-wire.png`, `breaking-bad.png`, `los-soprano.png`.
- No guardar videos como entregable del blog.

Estos posters marcan la referencia visual para futuros posters: cartel vertical 2:3, energia de cine 80/90, Rulo y Toni integrados en el universo de la obra.
